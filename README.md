## Mortal Kombat Plus

Material de estudo para desenvolvimento de um jogo de luta para o Mega Drive usando a SGDK me baseando no jogo Mortal Kombat

### Build

```shell
make SGDK_VERSION=2.11 compile
````

Rom em `out/rom.bin`

### Retroarch

```shell
make run
```
